🦌 deer-flow

开源长周期 SuperAgent 框架——自主完成需要几分钟到数小时的复杂任务，集成沙箱、记忆、工具与多子智能体编排于一体。

⭐ 63.7k Stars 🍴 8.3k Forks 📝 Python / TypeScript 📄 MIT License 🔗 bytedance/deer-flow
agent agentic-framework agentic-workflow deep-research super-agent langgraph sandbox long-term-memory

DeerFlow（Deep Exploration and Efficient Research Flow）是字节跳动开源的超级智能体框架，2026年2月发布 2.0 重写版本，基于 LangGraph 构建。它能够编排子智能体网络、持久化记忆系统和隔离沙箱，完成从深度研究到代码执行、内容创作的全链路复杂任务。凭借 Agent Skills 设计——用 Markdown 文件定义智能体能力模块——实现了"即插即用"的可扩展架构。

💡 为何值得关注：DeerFlow 代表了 2026 年 Agent 框架的新范式转变——从"单轮对话助手"升级为"可持续运行数小时的自主工作流引擎"。Agent Skills 设计让任何人都能用 Markdown 扩展智能体能力，无需深入框架代码；而 Context Engineering（上下文工程）模块解决了长任务中上下文爆炸的核心难题。

🚀 五大核心功能

🔧 Agent Skills（技能模块）

用 Markdown 文件定义能力，支持内置技能（研究、报告、幻灯片）和自定义技能 + MCP 服务器。仅按需加载，保持上下文精简。

🤖 Sub-Agents（子智能体）

主智能体动态生成子智能体，每个子智能体拥有独立上下文、工具和终止条件。支持并行执行，结果由主智能体综合整合。

📦 Sandbox（执行沙箱）

每个任务拥有隔离执行环境，支持 Local / Docker / Kubernetes 三种沙箱模式。文件系统结构清晰：uploads / workspace / outputs。

🧠 Context Engineering（上下文工程）

子智能体间上下文互相隔离；主动摘要压缩已完成任务；严格工具调用恢复机制，防止中间件中断错误。

💾 Long-Term Memory（长期记忆）

跨会话持久化记忆，自动记住用户画像、偏好与积累的知识。本地存储可控，自动去重避免重复积累。

📡 Message Gateway（消息网关）

集成 Telegram / Slack / Feishu / WeChat / WeCom 等主流 IM 渠道。支持 LangSmith / Langfuse 全链路追踪与可观测性。

⚡ 快速开始

推荐使用 Docker 部署，三步即可完成本地启动（需提前配置 LLM API Key）。
# 1. 克隆项目 git clone https://github.com/bytedance/deer-flow.git cd deer-flow # 2. 运行交互式配置向导（自动配置模型和环境） make setup # 3. Docker 启动（推荐生产模式） make docker-start # 访问 http://localhost:2026 即可开始使用
# Python 嵌入式客户端调用示例 from deerflow.client import DeerFlowClient client = DeerFlowClient() # 发送任务（流式输出） for event in client.stream("帮我研究 2026 年 AI Agent 框架趋势"): if event.type == "messages-tuple": print(event.data["content"]) # 列出可用模型 models = client.list_models()

🏗️ 技术架构

DeerFlow 2.0 基于 LangGraph + LangChain 重写，提供标准模式和实验性 Gateway 模式两种部署选项。
特性详情
核心框架LangGraph + LangChain（Python 70.1%，TypeScript 18.7%）
模型兼容性任何 OpenAI 兼容 API（GPT-4o / Claude / Gemini / 本地模型）
沙箱模式Local / Docker / Kubernetes 三档可选
IM 集成Telegram / Slack / Feishu / WeChat / WeCom
可观测性LangSmith / Langfuse 追踪集成
部署要求（推荐）生产：16 vCPU / 32GB RAM；开发：8 vCPU / 16GB RAM
模型无关 MIT 开源 LangGraph 驱动 多 IM 渠道

🎯 适合哪些场景

  • 深度研究任务：自动搜索、整合、分析多源信息，生成结构化研究报告
  • 代码自动化：在隔离沙箱中自主编写、调试、执行代码，处理复杂工程任务
  • 内容创作流水线：从调研到撰写文章、生成幻灯片、制作网页，全链路自动化
  • 企业 IM Bot：接入 Feishu / WeCom，让员工通过聊天界面触发复杂 Agent 工作流
  • 长周期自主任务：处理需要数小时完成的分布式、多步骤任务，无需人工干预

💡 核心亮点总结

  • Agent Skills = Markdown 文件：任何人都能用 Markdown 定义新的智能体能力，无需改动框架代码，极大降低扩展门槛
  • Context Engineering 模块：业界罕见的系统性解决方案——隔离子上下文 + 主动摘要压缩，让长任务不再"失忆"
  • 字节跳动背书：⭐63.7k 星，内部大规模使用验证，质量和稳定性有工业级保障
  • 开放生态：支持 MCP 服务器扩展 + 任意 OpenAI 兼容模型，不锁定特定供应商
