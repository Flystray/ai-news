MiroFish

简洁通用的群体智能引擎，预测万物

⭐ 55.7k 🍴 8.4k 📝 Python / Vue 📄 AGPL-3.0 🔗 666ghj/MiroFish
multi-agent-simulationknowledge-graphllmsagent-memoryfinancial-forecastingfuture-prediction

🎯 核心理念：在数字沙盒中"排练未来"，在无数次模拟中赢得决策。

Miro（镜像，映射真实世界的群体智能）+ Fish（鱼群，象征群体智能）

🚀 核心功能

  • 种子信息提取 - 从现实世界提取种子信息（突发新闻、政策草案、金融信号等）
  • 平行数字世界构建 - 自动构建高保真的平行数字世界
  • 多智能体模拟 - 数千个具有独立个性、长期记忆和行为逻辑的智能体自由互动
  • 上帝视角注入 - 从"上帝视角"动态注入变量，精确推演未来轨迹
  • 预测报告生成 - 自动生成详细的预测报告
  • 深度交互 - 与模拟世界中的任何智能体进行对话互动

⚙️ 技术亮点

5步工作流程：

  • 图构建 - 种子提取、个体/集体记忆注入、GraphRAG 构建
  • 环境设置 - 实体关系提取、角色生成（Persona）、智能体配置注入
  • 模拟 - 双平台并行模拟、自动解析预测需求、动态时间记忆更新
  • 报告生成 - ReportAgent 配备丰富工具集、深度互动后模拟环境
  • 深度交互 - 与模拟世界中的任何智能体聊天、与 ReportAgent 互动

核心技术栈：

  • GraphRAG - 知识图谱增强检索，支撑复杂关系推理
  • 多智能体系统 - 支持数千个智能体并发模拟
  • 长期记忆系统 - Zep Cloud 记忆管理
  • OASIS 引擎 - 基于 CAMEL-AI 的 OASIS 模拟引擎
  • 灵活LLM支持 - 支持任何 OpenAI SDK 格式的 LLM API

👥 适用人群

  • 政策制定者 - 零风险测试政策影响
  • 金融分析师 - 市场趋势预测
  • 舆情研究者 - 公共事件舆论走向模拟
  • 创意工作者 - 小说结局推演、创意探索

⚡ 快速上手

Docker 部署（推荐）：

docker compose up -d

源码部署：

# 1. 配置环境变量
cp .env.example .env

# 2. 一键安装依赖
npm run setup:all

# 3. 启动服务
npm run dev

前端：http://localhost:3000 | 后端：http://localhost:5001
