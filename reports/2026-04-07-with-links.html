GPT-6「土豆」正式曝光：4月14日发布，性能暴涨40%，200万Token超大上下文
OpenAI 下一代旗舰模型 GPT-6（内部代号 Spud，即"土豆"）被多位知情人士和泄露文档全方位曝光。据悉，GPT-6 预计将于 4月14日 正式发布，在代码编写、逻辑推理和智能体任务上相比 GPT-5.4 性能提升约 40%。技术规格方面，上下文窗口扩展至令人震撼的 200万 Token，原生多模态支持文本、音频、图像、视频的统一处理，无需依赖任何插件。更值得关注的是，OpenAI 为全力冲刺 GPT-6，已将产品团队更名为"AGI Deployment"（AGI 部署部），并暂停了 Sora 视频生成等非核心项目，体现了奥特曼对 AGI 的孤注一掷式战略押注。与此同时，GPT-Image 2 生图能力的短暂亮相也为 GPT-6 多模态功能奠定了伏笔。

DeepSeek V4 全面换装华为昇腾950PR：推理速度暴升35倍，国产算力里程碑
据 The Information、The Decoder 等权威科技媒体援引五位知情人士消息，DeepSeek 即将发布的下一代旗舰模型 V4 将完全基于华为昇腾950PR芯片运行，彻底摆脱对英伟达的依赖。这一重大转变背后，是 DeepSeek 团队与华为、寒武纪长达数月的深度合作——团队重写了模型底层代码，针对昇腾架构进行全面优化，最终实现推理速度提升 35倍、能耗降低 40%。华为昇腾950PR芯片规格为 1PFLOPS（FP8）/ 2PFLOPS（FP4），内存达 128GB，互联带宽 2TB/s。值得关注的是，DeepSeek 并未向英伟达、AMD 等国际厂商开放 V4 的测试权限。阿里、字节、腾讯等巨头已提前下单数十万颗华为昇腾芯片，国产算力订单供不应求，DeepSeek V4 预计本月底正式上线。

中国AI大模型调用量连续五周超越美国：千问3.6-Plus单日1.4万亿Token刷新全球纪录
OpenRouter 最新数据显示，上周（截至2026年4月6日）全球 AI 大模型总调用量为 27万亿 Token/周，环比增长 18.9%。其中中国模型周调用量达 12.96万亿 Token（环比增长 31.48%），美国仅为 3.03万亿 Token（环比增长 0.76%）——中国已连续五周领先全球。值得一提的是，阿里 Qwen3.6-Plus 编程大模型在 OpenRouter 单日调用量飙升 711% 至 1.4万亿 Token，刷新该平台全球单模型单日调用历史纪录。与此同时，字节跳动豆包大模型日均 Token 使用量突破 120万亿，三个月内翻倍，较 2024 年 5 月首次发布时增长超 1000 倍，成为继 OpenAI、Google 之后全球第三家日均 Token 消耗量突破 100 万亿的玩家。

Anthropic 于北京时间4月4日向订阅用户发出邮件通知，自该日起，Claude 订阅用户将无法再通过 OpenClaw 等第三方工具消耗订阅额度。导火索是有用户通过 $200/月 的 Claude Max 订阅，借助 OpenClaw 批量调用，用出了相当于 $5000 算力价值的资源。Anthropic 对此表示无法接受，认为这违反了服务条款。此举让"龙虾之父"（Claude Code 架构师 Boris）在 X 上公开表示沟通失败，并指责对方"抄功能"。与此形成对比的是，MiniMax 随即发声反对，强调其 Token 计划支持跨平台无限制使用，借机争夺用户。此事件标志着 AI 生态从开放协作走向封闭博弈的转折点。

全国首个具身智能机器人保洁员深圳上岗：AI具身智能行业标准6月落地
全国首个具身智能机器人保洁员在深圳正式上岗，标志着 AI 具身智能技术从实验室走向真实家庭场景。该机器人可执行家庭保洁、分拣、护理等任务，具备持续学习与场景自适应能力。值得关注的是，工信部已批准发布具身智能基准测试方法行业标准，将于 2026年6月1日 起正式实施，这是该领域首个国家行业标准落地，将有力推动具身智能商业化进程的规范化发展。专家指出，从单一家庭场景向商业写字楼、酒店、医院等复杂场景扩展，将是具身智能产业接下来的核心战场。

⚔️ GPT-6 与国产大模型的"四月决战"
4月14日或将成为2026年AI行业最重要的节点之一——GPT-6发布的同时，千问3.6-Plus已在全球调用量榜单登顶，DeepSeek V4也在本月底随时待命。这不是一次简单的版本迭代，而是一场关于"谁能定义下一代AI底座"的全球竞争。国产大模型第一次在规模应用层面实现对美国的持续超越，标志着全球AI竞争从"技术Demo秀"进入"规模化落地"的新阶段。
🇨🇳 算力自主化：从"不得不"到"真能打"
DeepSeek V4 完全切换华为昇腾，推理速度暴升35倍，这不是权宜之计，而是技术自主化真正成熟的信号。当国产芯片能让顶尖模型跑出比英伟达还强的推理效率，"卡脖子"叙事将被彻底重写。阿里、字节、腾讯抢购昇腾芯片的行为，已经用真金白银投票：国产算力生态已具备独立支撑顶级大模型的能力。
🔒 AI 生态的"围墙花园"时代正在开启
Anthropic 封杀 OpenClaw 的核心逻辑是：当 API 调用成本与订阅利润倒挂，开放生态就变成亏本生意。这一逻辑同样将影响整个行业——未来的 AI 巨头将更像 App Store，以生态管控为核心，而非以开放 API 换取增长。这对开发者和创业公司意味着更高的平台风险，也意味着打造独立底座能力的战略价值进一步凸显。
🌏 全球 AI 资本的"抢人"时代
英国抢 Anthropic、中国产学研联动、米哈游捐资交大——这一天的新闻背后，是全球对 AI 顶级人才和顶级企业的"地理争夺战"加速开打。美国收紧管制的同时，英国、中东、东南亚纷纷以更优惠条件招揽 AI 企业。AI 产业的全球化格局正在以肉眼可见的速度重构，中国在调用量、应用落地两个维度已形成先发优势，下一场战役是基础研究的话语权。
