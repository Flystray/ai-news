1. ReMe 新项目 2.7k⭐

⭐ 2.7k 📝 Python 🕐 活跃于 13 天前 📄 Apache-2.0

ReMe: Memory Management Kit for Agents — Remember Me, Refine Me. 专为 AI Agent 设计的记忆管理工具包，解决多轮对话记忆丢失问题。

创新点："文件即记忆"哲学——将 Agent 记忆存为可读可编辑的 Markdown 文件；向量检索（70%）+ BM25 关键词匹配（30%）混合检索；LoCoMo 基准 86.23%，超越 HiMem（80.71%）和 Zep（81.06%）
背景与问题：大语言模型上下文窗口有限，多轮对话记忆会随时间丢失——这是目前所有 Agent 应用的共同痛点。ReMe 由阿里 AgentScope 团队开发，提供两套方案：轻量级的 ReMeLight（基于 Markdown 文件）和完整版 ReMe（基于向量数据库）。

核心亮点：自动推理前钩子（pre_reasoning_hook）是 ReMe 的灵魂——在每次 AI 推理之前，系统自动压缩长工具输出、检测上下文是否超限、生成对话摘要并异步持久化记忆，整个流程对开发者完全透明，接入只需一行代码。

差异化优势：与 Mem0、Zep 等竞品依赖黑箱数据库不同，ReMe 将记忆以 Markdown 文件形式存储在工作目录中，人类可直接读取、编辑甚至用 Git 版本控制——真正实现"记忆可审计"。

2. smolagents 26.6k⭐

⭐ 26.6k 📝 Python 🕐 持续活跃 📄 Apache-2.0

🤗 smolagents: a barebones library for agents that think in code. 用代码思考的极简智能体框架，核心逻辑仅 ~1000 行。

创新点："代码即动作"架构——LLM 直接输出并执行 Python 代码而非 JSON 字典，相比行业通用方案减少 30% API 调用步骤；支持 E2B/Docker/Modal/WebAssembly 多沙箱安全执行；100+ LLM 模型支持
理念颠覆：传统 Agent 框架（LangChain、AutoGen 等）的工具调用：LLM 输出 JSON 字典 → 框架解析 → 调用函数。smolagents 的 CodeAgent：LLM 直接生成可执行 Python 代码 → 框架运行代码。

为什么更好：代码天然支持变量、循环、条件判断——可以在多步骤间共享状态，而 JSON 调用每步都是独立的。论文验证：在复杂任务中代码方式减少 30% 推理步骤（即 30% 的 API 费用）。

极简哲学：核心代码约 1000 行，但功能完整：多 Agent 层级、多沙箱隔离、支持 100+ 种 LLM（LiteLLM）、MCP Server 工具获取、甚至将 HuggingFace Space 当工具直接调用。

3. adk-python v1.29.0 · 18.9k⭐

⭐ 18.9k 📝 Python 🕐 2026年4月最新更新 📄 Apache-2.0

An open-source, code-first Python toolkit for building, evaluating, and deploying sophisticated AI agents. Google 官方 Agent 开发套件，每两周更新一次。

创新点：会话回溯（Rewind）——将 Agent 对话回滚到任意历史状态；A2A（Agent-to-Agent）协议支持多 Agent 跨服务通信；内置 adk eval 评估工具，将 Agent 测试从"凭感觉"变成"有数据"
v1.29.0 三大新特性（2026年4月）：
① 会话回溯（Rewind）：Agent 在复杂任务中决策出错时，不需要重新开始整个流程，可将对话状态回滚到之前任意节点——类似代码调试的"断点回退"。
② Vertex AI 代码执行沙箱：在 Google Cloud 托管的安全隔离环境中执行 Agent 生成的代码，解决本地执行安全隐患。
③ adk eval 评估工具：用标准化评估数据集对 Agent 决策链质量打分，这是主流 Agent 框架中极少见的原生评估能力。

生产力价值：深度集成 Google Cloud，支持 A2A 协议实现多 Agent 分布式协作，适合企业级自动化流程构建。
