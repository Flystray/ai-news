Miro（镜像，映射真实世界的群体智能）+ Fish（鱼群，象征群体智能）

📌 项目背景

MiroFish 是一个下一代 AI 预测引擎，由多智能体技术驱动，由盛大集团赞助开发。

传统的预测方法往往依赖历史数据和统计模型，而 MiroFish 采用了一种全新的思路——通过构建高保真的"平行数字世界"，让数千个具有独立个性、长期记忆和行为逻辑的智能体在其中自由互动，从而推演未来的可能走向。

💡 这种方法特别适用于复杂系统的预测，如社会舆情演变、金融市场波动、政策影响评估等。

🚀 核心功能详解

1. 种子信息提取

从现实世界提取关键"种子信息"作为模拟起点：突发新闻、政策草案、金融信号、社交媒体热点等。

2. 平行数字世界构建

自动构建高保真平行世界：社会关系网络、GraphRAG 知识图谱、个体/集体记忆系统、真实环境约束。

3. 多智能体模拟（核心能力）

数千个智能体并发运行，每个都有独立个性、基于 Zep Cloud 的长期记忆、符合角色的行为模式，自由互动演化。

4. 上帝视角注入

模拟过程中动态注入变量：突发事件、外部冲击、干预措施，观察对系统演化的影响。

5. 预测报告生成

ReportAgent 自动生成详细报告：多种未来场景、发生概率、关键影响因素、决策建议。

6. 深度交互

与模拟世界中的智能体对话、向 ReportAgent 提问、调整参数重新运行、对比不同假设结果。

⚙️ 技术架构

5步工作流程

  • 图构建 - 种子提取、记忆注入、GraphRAG 构建
  • 环境设置 - 实体关系提取、角色生成、智能体配置
  • 模拟 - 双平台并行、需求解析、动态记忆更新
  • 报告生成 - ReportAgent 工具集、深度分析
  • 深度交互 - 与智能体聊天、与 ReportAgent 互动

核心技术栈

技术说明
GraphRAG知识图谱增强检索，支撑复杂关系推理
多智能体系统数千智能体并发，独立状态和行为逻辑
长期记忆基于 Zep Cloud，智能体记住历史互动
OASIS 引擎基于 CAMEL-AI 的开源社会交互模拟引擎
灵活LLM支持 OpenAI SDK 格式，GPT-4/Claude/DeepSeek 等

🎬 应用场景

1. 宏观决策支持

政策制定者在零风险虚拟环境中"预演"政策效果，识别阻力点和风险区域。

2. 金融趋势预测

考虑市场情绪和群体行为，生成多种可能的走势场景，而非仅依赖技术指标。

3. 舆情演变分析

模拟信息传播和观点演化，识别关键意见领袖和转折点（如武汉大学舆情报告案例）。

4. 创意内容推演

作家探索小说情节不同发展，如《红楼梦》遗失结局模拟。

5. 产品发布预演

预测新产品发布后的市场反应，识别用户痛点和亮点。

⚡ 快速上手
