VibeVoice
微软出品 语音 AI ICLR 2026 Oral
⭐ 38.1k 🐍 Python 📄 MIT License 📅 2026-01 开源 ASR
微软研究院开源的前沿语音 AI 模型家族，包含长音频语音识别（ASR）和文本转语音（TTS）两大方向。核心突破在于：单次处理长达 60 分钟的音频，同时输出说话人身份、时间戳和转录文本，并被 ICLR 2026 接收为 Oral 论文。

💡 核心创新点

  • 采用连续语音分词器（7.5 Hz 超低帧率），在保持音质的同时大幅降低计算成本
  • 使用 next-token diffusion 机制——LLM 理解上下文语义，扩散头负责生成高保真声学细节，两者分工明确
  • VibeVoice-ASR 7B：单次 60 分钟音频处理，内置说话人分离（Speaker Diarization）和时间戳对齐
  • VibeVoice-Realtime 0.5B：首包延迟仅 ~300ms，已集成至 Hugging Face Transformers v5.3.0
  • TTS 支持 90 分钟多说话人合成（4 个说话人），包含即兴歌唱能力

🎯 使用场景

  • 会议记录自动化：上传 1 小时会议录音，直接输出"谁/何时/说了什么"的结构化纪要
  • 播客/有声书生产：多角色长篇内容合成，无需人工配音
  • 实时输入法：社区基于此已做出 Vibing 语音输入法（macOS/Windows），延迟极低
  • 跨语言内容制作：英文→中文语音，或跨语言歌唱合成

Hermes Agent
Nous Research 自我学习 Agent $5 VPS 可运行
⭐ 44.6k 🐍 Python 📄 MIT License 📅 v0.8.0 · 2026.04.08
Nous Research 出品的"会自我成长的 AI Agent"。区别于其他 Agent 框架只能执行预设任务，Hermes 内置了闭环学习机制：每完成一个复杂任务后，会自动创建新技能、强化记忆、优化已有工具，越用越智能。5 美元的 VPS 或 Serverless 平台即可部署，通过 Telegram/Discord 从手机远程控制。

💡 核心创新点

  • 自主技能创建：执行复杂任务后，Agent 会自动将"经验"提炼为可复用的技能，并写入 ACP 技能注册表
  • 跨会话持久记忆：使用 FTS5 全文索引，支持语义级跨会话记忆检索；Honcho 框架维护用户画像（辩证用户建模）
  • RL 训练闭环：内置 Atropos RL 环境，可批量生成工具调用轨迹，用于下一代工具调用模型的微调
  • 多平台统一网关：Telegram/Discord/Slack/WhatsApp/Email 等 6+ 平台，单一网关进程管理
  • 支持 200+ LLM（OpenRouter）无缝切换，无需改代码，含 DeepSeek/Kimi/GLM 等国产模型

🎯 使用场景

  • 个人 AI 管家：云端 24/7 运行，手机 Telegram 随时下发任务，Agent 后台自主完成
  • 企业自动化：每日报告生成、每周代码审计、数据库备份，cron 调度自然语言配置
  • 开发加速器：并行化子代理处理多个代码任务，支持 SSH 连接到远程开发机
  • AI 研究实验：批量轨迹生成 + RL 策略探索，加速工具调用模型研究

TimesFM 2.5
Google Research 时间序列基础模型 ICML 2024
⭐ 16k 🐍 Python 📅 最新版 2.5 · 2025.09 🔗 已集成 Google BigQuery
Google Research 推出的时间序列预测基础模型，思路类比 LLM：先用海量时序数据预训练一个通用大模型，推理时无需微调即可零样本预测任何时间序列。2.5 版本将参数从 500M 压缩到 200M，同时上下文长度提升 8 倍至 16,000——更小、更快、更强。

💡 核心创新点

  • Decoder-only 架构：把预测序列当成"下一个 token 续写"问题，开创了时间序列基础模型新范式
  • 零样本推理：不需要用目标领域数据微调，直接预测销售/天气/传感器等任意序列
  • 16k 超长上下文：可利用超过 1 年的日粒度历史数据进行预测
  • 连续量化预测：可选 30M 量化头，同时输出 10 个分位数（P10~P90），量化不确定性
  • Google Cloud 官方产品：已集成至 BigQuery ML，是首个进入主流云平台的开源时序基础模型

🎯 使用场景

  • 零售需求预测：无需历史样本，直接预测新 SKU 的销售量，解决冷启动问题
  • 云资源容量规划：预测服务器流量峰值，自动扩缩容决策
  • 金融时序分析：量化分位数预测给出置信区间，而非仅一个点预测
  • 工业设备监控：传感器异常预测，预测性维护（Predictive Maintenance）
  • 个人数据分析：200M 模型可在普通笔记本上运行，适合轻量本地部署
