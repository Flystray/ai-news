fikrikarim/parlor ⭐ 1.3k 🔥 本周新建

设备端实时多模态 AI 对话系统。使用 Google Gemma 4 E2B + Kokoro TTS，在本地机器上实现语音+视觉的实时 AI 对话交互。所有处理完全离线，端到端延迟仅 2.5-3 秒。

🎯 核心创新点
  • ✦免提操作 — 集成 Silero VAD 语音活动检测，说完即识别，无需按键触发
  • ✦随时打断 — 支持 Barge-in，可随时中断 AI 说话进行追问
  • ✦流式 TTS — 句子级语音合成，响应生成同时开始播放，几乎无等待感
  • ✦视觉+语音同步 — 边说话边展示摄像头画面，AI 同步理解并回应
⚙️ 技术架构

语音合成Kokoro TTS (Mac: MLX / Linux: ONNX)
语音检测Silero VAD
后端框架FastAPI + WebSocket

aiming-lab/MetaClaw ⭐ 3.4k 🧬 可进化

"Just talk to your agent — it learns and EVOLVES"。让 AI Agent 通过日常对话学习和进化的框架。受大脑学习方式启发，将每次对话转化为学习信号，支持云端 LoRA 微调，无需 GPU。

🎯 核心创新点
  • ✦对话即训练 — 无需专门训练数据，日常对话自动生成技能和 LoRA 权重
  • ✦跨会话记忆 — 5种记忆类型：情景/语义/偏好/项目状态/工作摘要
  • ✦云端训练 — 所有微调通过 Tinker/MinT API 完成，本地无需 GPU
  • ✦自动技能进化 — 会话结束后自动总结并生成新技能注入
⚙️ 技术架构
模块技术选型
训练算法GRPO + 在线 LoRA 微调
训练后端Tinker / MinT / Weaver
支持代理OpenClaw, CoPaw, IronClaw, ZeroClaw 等 8 种
记忆系统Contexture (v0.4.0 新增)
LLM 接口OpenAI / Anthropic / Kimi / Qwen 兼容
