## 1. [ReMe](https://github.com/agentscope-ai/ReMe)

**ReMe: Memory Management Kit for Agents — Remember Me, Refine Me.**

🕐 活跃于 13 天前 · ⭐ 2.7k · 📝 Python · 📄 Apache-2.0

**创新点**: "文件即记忆"哲学 — 把 Agent 记忆存成可读可编辑的 Markdown 文件；向量检索（70%）+ BM25 关键词匹配（30%）混合检索；LoCoMo 基准 86.23%，超越 HiMem（80.71%）和 Zep（81.06%）

ReMe（Remember Me, Refine Me）是阿里 AgentScope 团队推出的 AI Agent 记忆管理工具包，专门解决大语言模型上下文窗口有限、多轮对话记忆丢失的痛点。提供两套方案：轻量级的 ReMeLight（基于文件）和完整版 ReMe（基于向量），均支持语义搜索、工具记忆、程序记忆三类记忆管理。

**核心亮点**在于**自动推理前钩子（pre_reasoning_hook）**：在每次 AI 推理之前自动压缩长工具输出、检测上下文超限、生成摘要并异步持久化记忆——整个流程对应用层完全透明，接入只需一行代码。

与 Mem0、Zep 等竞品依赖黑箱数据库不同，ReMe 将记忆以 Markdown 文件形式存储在工作目录，人类可直接读取、编辑甚至用 Git 版本控制——真正实现"记忆可审计"。

**Topics**: `agent` · `memory` · `rag` · `llm` · `agentscope`

🔗 [GitHub](https://github.com/agentscope-ai/ReMe)

## 2. [smolagents](https://github.com/huggingface/smolagents)

**🤗 smolagents: a barebones library for agents that think in code.**

⭐ 26.6k · 📝 Python · 📄 Apache-2.0

**创新点**: "代码即动作"架构——LLM 直接输出并执行 Python 代码而非 JSON 字典，相比行业通用方案减少 30% API 调用步骤；核心逻辑仅 ~1000 行极简设计；支持 E2B/Docker/Modal/WebAssembly 多沙箱安全执行

Hugging Face 推出的极简 Agent 框架，核心主张是让 AI 以"代码的方式思考"：传统 Agent 框架让模型输出 JSON 字典来调用工具，而 smolagents 的 CodeAgent 让模型直接生成并执行 Python 代码片段。论文验证，这一设计在复杂任务中减少了 **30% 的推理步骤**（即 30% 的 LLM API 费用），原因在于代码天然支持变量、循环、条件判断，可在多步骤间共享状态。

框架极简但不简陋——支持 100+ 种 LLM（LiteLLM 集成）、MCP Server 工具获取、LangChain 工具复用，甚至将 HuggingFace Space 当工具直接调用。核心代码约 1000 行，鼓励开发者直接阅读修改，杜绝传统大框架的"黑盒焦虑"。

**Topics**: `agent` · `llm` · `code-generation` · `huggingface` · `mcp`

🔗 [GitHub](https://github.com/huggingface/smolagents)

## 3. [adk-python](https://github.com/google/adk-python)

**An open-source, code-first Python toolkit for building, evaluating, and deploying sophisticated AI agents with flexibility and control.**

⭐ 18.9k · 📝 Python · 📄 Apache-2.0

**创新点**: 会话回溯（Rewind）——将 Agent 对话回滚到任意历史状态；A2A（Agent-to-Agent）协议支持多 Agent 跨服务分布式通信；内置 adk eval 评估工具；Vertex AI 代码执行沙箱

Google 开源的官方 Agent 开发工具包，v1.29.0（2026年4月）新增三大特性：

- **会话回溯（Rewind）**：Agent 决策出错时可回滚到任意历史节点——类似代码调试器的"断点回退"，大幅降低 Agent 任务失败成本。
- **Vertex AI 代码执行沙箱**：在 Google Cloud 托管的安全隔离环境中执行 Agent 生成的代码，解决本地执行安全隐患。
- **adk eval 评估工具**：用标准化数据集对 Agent 决策链质量打分——这是主流 Agent 框架中极少见的原生评估能力，把 Agent 测试从"凭感觉"变成"有数据"。

**Topics**: `agent` · `google` · `gemini` · `multi-agent` · `mcp`

🔗 [GitHub](https://github.com/google/adk-python)
